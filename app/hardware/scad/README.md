# OpenSCAD Scripts (Archived Fallback)

These `.scad` files are **archived fallback** scripts. build123d (`hardware/cad/`) is the primary backend for all parts.

OpenSCAD scripts are kept for reference and as a fallback when build123d is unavailable. They are **not actively maintained** — changes should be made to the build123d scripts first.

The `primary_backend` field in `hardware/parts.json` controls which backend `render.py` uses per part.
