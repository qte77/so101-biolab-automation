# OpenSCAD Scripts (Archived Fallback)

These `.scad` files are **archived fallback** scripts. CadQuery (`hardware/cad/`) is the primary backend for all parts.

OpenSCAD scripts are kept for reference and as a fallback when CadQuery is unavailable (e.g., environments where Python 3.12 + CadQuery cannot be installed). They are **not actively maintained** — changes should be made to the CadQuery scripts first.

The `primary_backend` field in `hardware/parts.json` controls which backend `render.py` uses per part.
